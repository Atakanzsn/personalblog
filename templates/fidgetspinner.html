STRES ÇARKI

Gerildiğinizde sizi yumuş yumuş yapacak bir oyun, boşluk tuşunu kırmayın sakın!

Kullanımı:

Boşluk tuşu ile döndürebilirsiniz.

Kodlar:

            from turtle import *

            state = {"turn": 0}

            def spinner():
                "draw spinner"
                clear()
                angle = state["turn"] / 10
                right(angle)
                forward(100)
                dot(120,"red")
                back(100)
                right(120)
                forward(100)
                dot(120,"green")
                back(100)
                right(120)
                forward(100)
                dot(120,"blue")
                back(100)
                right(120)
                update()
            def animate():
                "animate"
                if state["turn"] > 0:
                    state["turn"] -= 1

                spinner()
                ontimer(animate, 20)

            def flick():
                "flick"
                state["turn"] += 10

            setup(420,420,370,0)
            hideturtle()
            tracer(False)
            width(20)
            onkey(flick, "space")
            listen()
            animate()
